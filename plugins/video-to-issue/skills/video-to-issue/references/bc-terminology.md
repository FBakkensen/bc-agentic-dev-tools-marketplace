# BC Terminology to Extract

When analyzing videos, identify and extract these Business Central concepts:

| Category | Terms to Look For |
|----------|-------------------|
| UI Elements | Pages, FactBoxes, Actions, Fields, Parts, Subpages |
| Navigation | Menu paths, Search bar usage, Tell Me, Role Center |
| Processes | Posting, Validation, Calculation, Release, Reopen |
| Technical (rare) | Codeunits, XMLports, Tables, Records, Keys |

Capture navigation paths as shown in the video (e.g., "Search for 'Sales Orders' → Open card → Actions → Post").
